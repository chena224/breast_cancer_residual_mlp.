# 基于残差MLP的乳腺癌肿瘤分类模型

本项目使用 PyTorch 实现带残差连接的深度神经网络，对乳腺癌数据集进行二分类（良性/恶性）。
项目采用标准化深度学习训练流程，结构完整、可复现、可直接用于学习与求职展示。

## 主要功能
- 残差连接模块，提升深层网络训练稳定性
- 支持 BatchNorm / LayerNorm、Dropout 等正则化方式
- 完整训练框架：早停策略、梯度裁剪、余弦退火学习率调度
- 训练曲线、混淆矩阵、错误案例可视化
- 固定随机种子，保证实验可复现

## 任务说明
- 任务类型：二分类（良性/恶性）
- 数据类型：医学表格数据
- 特征数量：30 维
- 模型结构：残差连接 MLP

## 技术栈

- PyTorch
- scikit-learn
